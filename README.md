# 🌐 Proyecto Web SEO – Programación con Entornos de Trabajo

## 📄 Descripción general
Este proyecto consiste en el desarrollo de un sitio web estático enfocado en la correcta implementación de **buenas prácticas de desarrollo front-end** y **optimización SEO on-page**.

El objetivo principal fue construir una web semánticamente estructurada, accesible y optimizada para buscadores, aplicando criterios técnicos utilizados en entornos profesionales.

---

## 🎯 Objetivos del proyecto
- Aplicar principios de **SEO on-page** en un sitio real
- Implementar una **estructura semántica clara y accesible**
- Optimizar el contenido para mejorar su **indexación en buscadores**
- Desarrollar una web **responsive** adaptable a distintos dispositivos
- Incorporar **buenas prácticas técnicas** de desarrollo web

---

## 🛠️ Tecnologías utilizadas
- **HTML5** → Estructura del sitio y semántica
- **CSS3** → Estilos personalizados
- **Bootstrap** → Sistema de grillas y responsive design

---

## 🔍 Implementación SEO
Se aplicaron múltiples técnicas de optimización SEO on-page para mejorar la visibilidad del sitio en motores de búsqueda:

### Metaetiquetas
- Uso de **`<title>`** optimizado con palabras clave relevantes
- Implementación de **meta description** clara y atractiva
- Configuración de **meta robots** para controlar indexación

### Estructura de headings
- Jerarquía correcta:
  - `H1` → Título principal (único por página)
  - `H2` → Secciones principales
  - `H3` → Subniveles de contenido
- Mejora de la **legibilidad** y comprensión del contenido por buscadores

### Keywords
- Selección estratégica de palabras clave
- Distribución natural dentro del contenido
- Control de densidad para evitar sobreoptimización

### Imágenes
- Uso de atributos **ALT descriptivos**
- Mejora en accesibilidad y posicionamiento en búsquedas de imágenes

## ✅ Buenas prácticas aplicadas
- Uso de **etiquetas semánticas HTML5**:
  - `<header>`, `<nav>`, `<main>`, `<section>`, `<article>`, `<footer>`
- URLs amigables (naming claro y descriptivo)
- Separación de responsabilidades (estructura vs estilos)
- Código limpio y organizado
- Accesibilidad básica implementada

---

## 🤖 Archivos SEO técnicos

### robots.txt
- Configuración para permitir o bloquear el acceso a determinadas rutas
- Simulación de restricciones a carpetas no relevantes (ej: `/img/`)

### sitemap.xml
- Listado estructurado de páginas del sitio
- Priorización de URLs
- Mejora del rastreo por parte de motores de búsqueda

---

## 📱 Responsive Design
El sitio fue desarrollado con un enfoque **mobile-first**, garantizando compatibilidad en:

- 📱 Dispositivos móviles

- Breakpoints adaptativos

- Minimización de recursos innecesarios
- Carga eficiente del contenido
